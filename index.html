PROTOTYPE

The object I have chosen to redesign is the Calculator. When I’m working in the wood shop, I find myself using the calculator to calculate measurements. Regular calculators are big and hard to use because in Canada we still use the imperial (INCH) system and material (wood) is still sold in imperial.

Woodworkers in the shop are using a app that calculates fractions making adding and subtracting fractions easier as they are presented in fractions and not in long decimals.

The issue is I don’t like using my phone in my workspace because it is distracting and I don’t want to break it that’s why I want to make a Fractional Calculator for the woodworker in the physical form.
